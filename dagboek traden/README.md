# Dagboek Traden — XAUUSD (Gold)

Dit dagboek houdt alle trades en leermomenten bij.

## Structuur
- `trades/` — Individuele trade entries (YYYY-MM-DD_trade.md)
- `lessen.md` — Gecumuleerde lessen en systeemverbeteringen
- `statistieken.md` — Winrate, R-multiples, maandoverzicht
